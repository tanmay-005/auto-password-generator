# Password Generator

## Overview

This project is a secure password generator written in Python.
It allows users to generate strong passwords with customizable options such as uppercase letters, lowercase letters, digits, and special characters.

## Features

* User-defined password length
* Option to include:

  * Uppercase letters
  * Lowercase letters
  * Numbers
  * Special characters
* Uses Python's **secrets** module for cryptographically secure password generation

## Technologies Used

* Python
* string module
* secrets module

## How to Run

Run the program using:

python password_generator.py

Then follow the prompts to generate a secure password.
